- 스토리지

1) 장기적으로 저장하고 보관하는 역할
2) 사용자의 파일, 응용 프로그램, 운영 체제등을 포함

종류)

- 하드 디스크 드라이브 (HDD) : 부품을 사용하여 데이터를 저장
- 솔리드 스테이트 드라이브 (SDD) : 반도체 기술을 사용하여 데이터를 저장, HDD보다 빠르고 내구성이 높다
- 외장 하드 드라이브, USB 등 이외 다양한 형태의 스토리지 장치가 있다.

- 메모리

1) 현재 작업중인 데이터와 명령을 일시적으로 저장하는 역할
2) CPU가 작업을 처리할 때 필요한 데이터와 명령어를 빠르게 접근할 수 있도록 도와준다.
3) 읽고 쓰는 속도가 매우 빠르기 때문에 CPU가 신속하게 작업을 수행할 수 있다.


종류)

- 랜덤 엑세스 메모리(RAM) : 주로 시스템의 작동 중 사용, 전원이 꺼지면 데이터가 사라짐
- 캐시 메모리 : CPU 근처에 위치, 빠른 액세스를 제공하는 작은 용량의 메모리

정리)
스토리지 - 데이터 장기적 보존 가능
메모리 - 작업에 필요한 데이터를 빠르게 접근할 수 있도록 지원
